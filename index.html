Password Generator

To create a password generator, first you have to create an index.html. File inside the file you have to create a form and to make sure, that the user can enter any type of password such us (lowercase letters, up case letters, numbers and symbols). In this tutorial we gone use the select tag to arrow the user to enter the length of any type of character and a generate button for generate the new password.

1.index.html

After create the index.html file you have to create a style.css file. Inside this file is just to input the style for the html document such us (form, select, labels and paragraph). Warning! If the user click on the new password, the cursor will automatically select the new password. You will now more about this functionality in the next paragraph.

2.style.css

Finally, you have to create a script.js file. Inside this file, you have to get data from the html form and after you have to create an array. Inside the array, you have to enter all kind of data such us (lower case letters, up case letters, numbers and symbols). By using this method, the user can select the length of the characters in the html form and JavaScript will slice data inside the array and print the new password. before printing the new password JavaScript sort the array data as he want and numbers are converted in MS(milliseconds) so that if the user generate a new password , and click again to the generate button the password will be automatically changed because it use MS(milliseconds) counting. Lastly, if you click of the new password it automatically copied because the text has been select by the cursor.
